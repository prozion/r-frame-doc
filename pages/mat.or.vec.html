mat.or.vec R Documentation

Create a Matrix or a Vector

Description

mat.or.vec creates an nr by nc zero matrix if nc is greater than 1, and a zero vector of length nr if nc equals 1.

Usage

mat.or.vec(nr, nc)

Arguments

nr, nc

numbers of rows and columns.

Examples

mat.or.vec(3, 1)
mat.or.vec(3, 2)
